# ACCESS missionA
This is the project for ACCESS companay. 

## Requriments
The projecet is based on Django,  
  
so we have to install Django at first.  
  
Install [Django](https://github.com/django/django)  

`pip install django`  

## Start  


`python manage.py runserver [port]`
